# 🚀 Webisso Open Source

Welcome to the official GitHub home of **Webisso**. We build modern, open-source tools and libraries designed to simplify digital workflows, enhance accessibility, and empower developers.

---

## 🛠 Featured Projects

Our ecosystem focuses on automation, multilingual support, and developer utilities.

### 🤖 Automation & AI
* **[file-translator](https://github.com/Webisso/file-translator)**: An open-source file translation UI powered by AI. Seamlessly integrates with **n8n** to handle document localization at scale.

### 📰 Content & Feed Management
* **[swipe-rss](https://github.com/Webisso/swipe-rss)**: A curated, multi-language RSS feed library in OPML format. Manage and subscribe to global feeds with a single URL.

### 🔧 Utility Tools
* **[qr-generator](https://github.com/Webisso/qr-generator)**: A modern, fully client-side QR code generator. Privacy-focused, multi-language, and lightning-fast.
* **[playing-cards](https://github.com/Webisso/playing-cards)**: A high-quality collection of open-source playing card assets in PNG and SVG formats for game developers.

### 🛡️ Infrastructure & Networking
* **[wireguard-apple](https://github.com/Webisso/wireguard-apple)**: Our maintained mirror/fork of the official WireGuard implementation for Apple platforms.

---

## 🌐 Vision & Values

At Webisso, we believe in the power of the open web. Our projects aim to be:
- **Global-First:** Built-in multi-language support is a standard, not an afterthought.
- **Privacy-Centric:** Favoring client-side execution where possible (like our QR generator).
- **Integratable:** Designed to work within modern stacks like n8n and existing RSS workflows.
